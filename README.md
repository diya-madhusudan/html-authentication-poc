# HTML Authentication POC

This project is a simple HTML-based authentication system created using only basic HTML structure and anchor tag redirections.

## Pages Included

- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features

- Navigation between pages using anchor tags
- No CSS used
- No JavaScript used
- Basic HTML structure only

## How to Run

1. Download or clone the repository
2. Open login.html in any browser
3. Test all page redirections
